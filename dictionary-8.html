Example 1

Input:

# pop method
d = {"Venky": 100, "Viky": 200, "Sendhil": 300, "Mani": 400}
d.pop()

Output:

TypeError: pop expected at least 1 argument, got 0

Example 2

Input:

v = d.pop("Sendhil")
print(v)
print(type(v))
print(d)

Output:

300
<class 'int'>
{'Venky': 100, 'Viky': 200, 'Mani': 400}

Example 5

Input:

# popitem()
d = {"Venky": 100, "Viky": 200, "Sendhil": 300, "Mani": 400}
v = d.popitem() # returns a tuple of last key_value pair
print(v)
print(type(v))
print(d)

Output:

('Mani', 400)
<class 'tuple'>
{'Venky': 100, 'Viky': 200, 'Sendhil': 300}
