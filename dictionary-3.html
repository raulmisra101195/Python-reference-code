Example 1

Input:

{'A': 100, 'B': 20, 'C': 30}
d1 = {'A': [10,20,30], 'B': 20, 'C': 30}
d2 = d1.copy()
print(d1)
print(d2)
d1['A'][1] = 100
print(d1)
print(d2)

Output:

# to overcome this, u go to deepcopy
{'A': [10, 20, 30], 'B': 20, 'C': 30}
{'A': [10, 20, 30], 'B': 20, 'C': 30}
{'A': [10, 100, 30], 'B': 20, 'C': 30}
{'A': [10, 100, 30], 'B': 20, 'C': 30}

Example 2

Input:

import copy
d1 = {'A': [10,20,30], 'B': 20, 'C': 30}
d2 = copy.deepcopy(d1)
print(d1)
print(d2)
d1['A'][1] = 100
print(d1)
print(d2)

Output:

{'A': [10, 20, 30], 'B': 20, 'C': 30}
{'A': [10, 20, 30], 'B': 20, 'C': 30}
{'A': [10, 100, 30], 'B': 20, 'C': 30}
{'A': [10, 20, 30], 'B': 20, 'C': 30}

Example 3

Input:

d = {'a': 10, 'b': 20, 'c': 30}
d_2 = dict(d)
print(d)
print(d_2)
d['a'] = 100
print(d)
print(d_2)

Output:

{'a': 10, 'b': 20, 'c': 30}
{'a': 10, 'b': 20, 'c': 30}
{'a': 100, 'b': 20, 'c': 30}
{'a': 10, 'b': 20, 'c': 30}
